# 1C Processor

## Описание

Внешняя обработка (.epf), содержащая табличную часть и набор инструментов для работы с данными: генерация, поиск дублей, сортировка и обработка строк.

---

## Функциональность

### 1. Заполнение таблицы

* Генерация строк по заданным параметрам:

  * количество строк
  * длина строки
* Используются случайные значения

---

### 2. Поиск дублей

Реализовано несколько способов:

* **Через цикл (Соответствие)**

  * группировка значений
  * сбор номеров строк

* **Через запрос (язык запросов 1С)**

  * использование временной таблицы
  * `СГРУППИРОВАТЬ` + `ИМЕЮЩИЕ`

Вывод:

* значения-дубли
* список строк, где они встречаются
* время выполнения каждого метода

---

### 3. Удаление дублей

* оставляется первая встреча значения
* остальные строки удаляются

---

### 4. Сортировка

* сортировка по полю
* переключение:

  * по возрастанию
  * по убыванию

---

### 5. Обработка строк

* инверсия строки (например: `1234 → 4321`)

---

### 6. Подсветка данных

* строки, содержащие заданную подстроку, выделяются цветом
* реализовано через оформление формы

---

## Используемые подходы

* Работа с `ТаблицаЗначений`
* Использование `Соответствие` для группировки
* Язык запросов 1С (включая временные таблицы)
* Разделение:

  * данные (`Объект`)
  * интерфейс (`Элементы`)
* Управление состоянием через переменные формы
